# Спецификация лендинга — Dmitrii Ivanov

## 1. Цель и аудитория

- **Цель:** получать офферы на FT-позицию Python Backend Developer.
- **Аудитория:** рекрутеры (основная), технические специалисты (вторичная).
- **Язык:** только RU. EN — отдельная фаза после утверждения.
- **Главный CTA:** написать в Telegram → `https://t.me/dm1tryivanov`.
- **Время на странице:** 30–90 секунд, нужно за это время «зацепить» и дать связаться.

## 2. Структура (single-page, 5 секций)

1. **Hero** — имя, роль, 1-строчный подзаголовок, локация/доступность, primary CTA «Написать в Telegram», secondary «Скачать PDF».
2. **Обо мне** — короткий абзац (3–4 строки) из секции «Обо мне» в PDF.
3. **Стек** — теги, сгруппированы как в PDF (Языки / Фреймворки / БД / Интеграции / Инфра / Качество).
4. **Опыт работы** — 3 карточки в обратном хронологическом порядке (Web Design Sun → Dev Symfony → Guider). Каждая: компания, роль, период, теги стека, буллеты с метриками из PDF.
5. **Контакты + Footer** — Telegram, Email, LinkedIn, Phone, языки (RU/EN/ES), приписка про UTC+3 / готовность к интервью. В футере: имя, год, переключатель темы.

Всё содержимое — дословно из PDF, без выдуманных деталей.

## 3. Дизайн

- **Референс:** [Xurya Manufacture Landing](https://dribbble.com/shots/24874505-Xurya-Manufacture-Landing-Page) — крупная типографика, чистая сетка, спокойный акцент, ощущение «продукт, а не портфолио».
- **Темы:** обе (тёмная по умолчанию + светлая), переключатель в футере, выбор сохраняется в `localStorage`.
- **Акцентный цвет:** на этапе дизайна предложу 1–2 варианта.
- **Типографика:** sans-serif для текста (Inter/Geist), mono для тегов и цифр (JetBrains Mono).
- **Анимации:** сдержанные fade/slide на скролле. Без параллакса и автоплея.
- **Адаптив:** mobile-first, 360 / 768 / 1024 / 1280.

## 4. Технические решения

- **Стек:** Vite + React 18 + TypeScript + Tailwind CSS.
- **Структура:** один маршрут, секции — компоненты в `src/sections/*`, контент — типизированные данные в `src/content/*.ts`.
- **PDF-резюме:** статикой в `public/`, ссылка из Hero и футера.
- **Без backend-а, без форм, без аналитики, без SEO-обвязки** на этой фазе.
- **A11y:** семантические теги, контраст AA+, фокус-стейты, навигация с клавиатуры. Внешние ссылки — `rel="noopener noreferrer"`.
- **Деплой:** Vercel (бесплатный план, автодеплой из Git, кастомный домен опционально).

## 5. Definition of Done

- [ ] 5 секций реализованы, контент сверен с PDF.

- [ ] Прошёл `react-reviewer` (хуки, perf, a11y) и `security-reviewer` (внешние ссылки, утечки данных).
